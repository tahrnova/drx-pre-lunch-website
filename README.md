.# DRX

**Smart System, Smart Care**

DRX is a healthcare workflow automation platform developed by **TahrNova**. The platform is designed to help hospitals, clinics, diagnostic centers, laboratories, and healthcare teams streamline operations through digital forms, workflow tracking, smart alerts, analytics, and centralized management.

## Features

* Pre-launch registration system
* Demo request management
* Smart workflow alerts
* Digital intake forms
* Lead scoring and qualification
* Analytics dashboard
* Admin login and management portal
* AI-powered product assistant
* Email notifications
* Supabase database integration

## Technology Stack

### Frontend

* Next.js
* React
* Tailwind CSS

### Backend

* Node.js
* Express.js
* JWT Authentication
* Nodemailer

### Database

* Supabase
* PostgreSQL
